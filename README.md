# StudySphere - Collaborative Learning Network

StudySphere is a full-stack collaborative learning network scaffolded with:

- Java 21, Spring Boot 3.5, Maven
- Spring Security, JWT, Spring Data JPA, Hibernate
- MySQL 8
- React, Vite, Bootstrap 5, Axios, React Router

## Project Structure

```text
StudySphere/
  backend/
    pom.xml
    src/main/java/com/studysphere/
    src/main/resources/application.properties
  frontend/
    package.json
    vite.config.js
    index.html
    src/
  database/
    schema.sql
```
